# Static Visual Assets

Local assets for replacing Stitch remote image and Material Symbols usage.

Recommended page paths:

- Home hero: `/static/images/home-hero-study.svg`
- Window seat card: `/static/images/seat-window.svg`
- Quiet pod seat card: `/static/images/seat-quiet-pod.svg`
- Access dark background: `/static/images/access-bg.svg`
- VIP wood texture: `/static/images/vip-wood-texture.svg`
- Avatar fallback: `/static/images/avatar-placeholder.svg`
- Icons: `/static/icons/ic-*.svg`

All icons use a 24x24 viewBox. Most icons inherit `currentColor`; filled symbols such as star and bolt use `fill="currentColor"`.
